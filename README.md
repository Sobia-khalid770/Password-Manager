# Password Manager (C++)

A simple **console-based password manager** written in C++ that can:

- Generate strong random passwords
- Save passwords along with service name and username
- View all saved passwords

This project is beginner-friendly and demonstrates **file handling, random password generation, and basic console menus** in C++.

---

## Features

1. **Generate Passwords**  
   - Uses letters (upper & lower case), numbers, and symbols (`!@#$%^&*`)  
   - Password length is user-defined

2. **Save Passwords**  
   - Stores passwords in a local text file (`passwords.txt`)  
   - Saves **Service name**, **Username**, and **Password**  

3. **View Saved Passwords**  
   - Reads the `passwords.txt` file  
   - Displays all saved credentials in the console

---

## How to Use

1. Clone or download the repository.  
2. Open the project in your favorite C++ IDE (like Visual Studio).  
3. Compile and run the program.  
4. Use the menu to:
   - Generate a new password
   - Save a password with service name and username
   - View all saved passwords
   - Exit the program
